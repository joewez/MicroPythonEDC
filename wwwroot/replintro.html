Introduction to the Read-Evaluate-Print-Loop (REPL)

The MicroPython REPL is a way of interacting with your hardware to execute MicroPython code. To use it, simply type in your command at the >>> prompt and press Enter. Your code will immediately be interpreted and executed.

For instance, enter this command... print('Hello World!') and view the generated output...

>>> print('Hello World!')
Hello World!

You can enter Python expressions...

>>> 13 * 56
728

or

>>> 3 * "Ha "
'Ha Ha Ha '

or import code in an existing module...

>>> import machine

The REPL is a great place to experiment and try out new commands and code. Here are some tips about using it...

  • You can use the UP arrow key to go to previously entered commands
  • The prompt will automatically indent the code when it is needed
  • You can enter CTRL+E to enter 'paste' mode where you can paste in code directly

At times, you will notice that the device does not seem to respond to keyboard input. This is usually because it is currently busy running a script. To interrupt the script you can try typing Ctrl-C. This will usually be able to interrupt a runnning application.
