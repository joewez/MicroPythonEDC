Ampy File Manager

If you are on a Windows machine, you can use a graphical application to manage your Pocket Cloud device.

Installing The Application

  1. Install Python 3
    https://python.org/
  2. Install Adafruit AMPY. At a command prompt enter:
    pip install adafruit-ampy
  3. Unzip the application from
    http://wezensky.no-ip.org/shared/afm.zip
  4. Run the main executable AmpyFileManager.exe
