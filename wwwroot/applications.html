Shield Applications

Since the Pocket Cloud is a general-purpose type of device, it can be re-programmed to do many different and various things. It is designed to work as-is, or with one or more "shield" attachments that plug in to the top of the device. Below are a list of valid commands to place in the main.py file broken down by shield that is being used.

Please note that the shields can be plugged in two ways. To verify the correct orientation, just check for any writing on the shield. The writing should be oriented such that the bottom is facing the USB connector.

No Shields

import uhttp

FTP Server for Devices Files

import uftpd

In order for the Web or File server to work, the device should be part of a network at this point. Either as a client to an existing Wifi network or when configured as an access point itself. See the information on controlling wifi connections here.

Neopixel Shield

Rainbow (Slow changing)

from drivers import neopixelstrip
nps = neopixelstrip.NeoPixelStrip(pin_number=4, pixel_count=7)
nps.rainbow_animate(cycles=0, delay=1000)

Rainbow (Fast changing)

from drivers import neopixelstrip
nps = neopixelstrip.NeoPixelStrip(4, 7)
nps.rainbow_animate(0, 50)

Solid Color (Pre-Defined)

from drivers import neopixelstrip
nps = neopixelstrip.NeoPixelStrip(4, 7)
nps.blue()
nps.red()
nps.green()
nps.white()

Solid Color (Custom)

from drivers import neopixelstrip
nps = neopixelstrip.NeoPixelStrip(4, 7)
nps.color(<red>, <green>, <blue>)

Where each value must be a number between 0 and 255.


Individual Pixel Control

from drivers import neopixelstrip
nps = neopixelstrip.NeoPixelStrip(4, 7)
nps.set_pixel(index=0, red=0, green=0, blue=0)
nps.set_pixel(1, 128, 128, 128)
