General Usage

Here are some basic commands to demonstrate how to operate and configure the device.

Changing the Main Program

To edit the main.py file simply enter:

>>> import editmain

This will open a full-screen editor with the current main.py file loaded. Here are some quick keys you may need...

  • Ctrl-S - Save File
  • Ctrl-Q - Quit Editor
  • Ctrl-E - Redraw Screen
  • Ctrl-Z - Undo
To learn about all of the editing commands available, please see this page.


File Commands

The Personal Pocket Cloud has a small amount of flash memory that can be used to hold MicroPython scripts and other data.

All the following commands require that you import the util module.

>>> import util as u

View The Files In the Root Directory

>>> u.ls()

View The Files In A Specific Directory

>>> u.ls('/wwwroot')

View The Contents Of A File

>>> u.cat('main.py')

Edit A File

>>> u.edit('/wwwroot/index.html')

Get Free Space

>>> u.df()


WiFi Commands

Note: The ESP8266 WiFi chip used by the Pocket Cloud will automatically retain it's last WiFi settings even after it is reset or even re-powered. You have to use the following commands to change the settings directly.

All the following commands require that you import the wifi module.

>>> import wifi

Connect to a WiFi Network

At the MicroPython command prompt enter the command:

>>> wifi.connect('ssid', 'password')

Using the appropriate credentials for your situation. Note that case does matter for both parameters. For example:

>>> wifi.connect('HOME_WIFI', 'ABDCEFG')

Disconnect from a WiFi Network

At the MicroPython command prompt enter the commands:

>>> wifi.disconnect()

Create a WiFi Access Point

At the MicroPython command prompt enter the commands:

>>> wifi.access_point('ssid', <'password'>)

Using the credentials that you wish. Note that case does matter for both parameters and the password must be a minimal length (or no parameter for no password).

Disable all WiFi

At the MicroPython command prompt enter the commands:

>>> wifi.none()


System Commands

Check Memory

>>> import gc
>>> gc.mem_free()

List Available Modules

>>> help('modules')

Turn Off Wifi Debug Messages

>>> import esp
>>> esp.osdebug(None)
